开发 Button 组件 ​

接下来我们来开发第一个组件，Button 组件。在开发组件之前，我们先设计一下 Button 组件的 API。主要是参考 ElementPlus 的 Button 组件设计。

一、组件的属性 Attributes ​

属性名说明类型默认值
size尺寸enum - 'large' | 'default' | 'small'—
type类型enum - 'primary' | 'success' | 'warning' | 'danger' | 'info'—
plain是否为朴素按钮booleanfalse
round是否为圆角按钮booleanfalse
circle是否为圆形按钮booleanfalse
loading是否为加载中状态booleanfalse
loading-icon自定义加载中状态图标组件string / ComponentLoading
disabled按钮是否为禁用状态booleanfalse
icon图标组件string—
autofocus原生 autofocus属性booleanfalse
native-type原生 type 属性enum - 'button' | 'submit' | 'reset'button

二、组件的插槽 Slots ​

插槽名说明
default自定义按钮内容
loading自定义加载中状态组件
icon自定义图标组件

三、组件的导出 Exposes ​

属性名说明类型
ref组件的引用object - Ref<HTMLButtonElement>
size按钮尺寸object - ComputedRef<'' | 'small' | 'default' | 'large'>
type按钮类型object - ComputedRef<'' | 'primary' | 'success' | 'warning' | 'danger' | 'info'>
disabled按钮已禁用object - ComputedRef<boolean>

四、组件的开发 ​

根据组件的 API 设计，我们先写出组件的类型文件，如下：

js
export type ButtonType = 'primary' | 'success' | 'warning' | 'danger' | 'info'
export type ButtonSize = 'large' | 'small'
export type NativeType = 'button' | 'submit' | 'reset'

export interface ButtonProps {
  type?: ButtonType
  size?: ButtonSize
  plain?: boolean
  round?: boolean
  circle?: boolean
  disabled?: boolean
  nativeType?: NativeType
  autofocus?: boolean
  icon?: string
  loading?: boolean
}
export interface ButtonInstance {
  ref: HTMLButtonElement
}
export type ButtonType = 'primary' | 'success' | 'warning' | 'danger' | 'info'
export type ButtonSize = 'large' | 'small'
export type NativeType = 'button' | 'submit' | 'reset'

export interface ButtonProps {
  type?: ButtonType
  size?: ButtonSize
  plain?: boolean
  round?: boolean
  circle?: boolean
  disabled?: boolean
  nativeType?: NativeType
  autofocus?: boolean
  icon?: string
  loading?: boolean
}
export interface ButtonInstance {
  ref: HTMLButtonElement
}

接着，我们创建组件的主体文件 index.vue，

html
<template>
  <button
    ref="_ref"
    class="au-button"
    :class="{
      [`au-button--${type}`]: type,
      [`au-button--${size}`]: size,
      'is-plain': plain,
      'is-round': round,
      'is-circle': circle,
      'is-disabled': disabled,
      'is-loading': loading
    }"
    :disabled="disabled || loading"
    :autofocus="autofocus"
    :type="nativeType"
  >
    <span>
      <slot />
    </span>
  </button>
</template>
<script setup lang="ts">
import { ref } from 'vue'
import type { ButtonProps } from './types'
defineOptions({
  name: 'auButton'
})

withDefaults(defineProps<ButtonProps>(), {
  nativeType: 'button'
})

const _ref = ref<HTMLButtonElement>()

defineExpose({
  ref: _ref
})
</script>

这里通过覆盖类名的变量，来实现样式的覆盖。比如，按钮为朴素按钮，命中的 class 类名为 .au-button--$(val).is-plain，那么按钮颜色对应的变量 --au-button-text-color 的值，就会变成 var(--au-color-$(val))，从而实现样式的覆盖。
