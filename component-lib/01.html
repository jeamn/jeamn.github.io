TIP

发布日期：2023-08-20

公司一般都会有属于自己的基础组件库，比如阿里的 Ant Design、Ant Design Mobile，腾讯的 TDesign 等等，这些组件库都是基于 React/Vue 的，那么我们自己也可以写一个基于 React/Vue 的组件库，这样就可以在公司内部使用，从而提高开发效率，保证公司产品的 UI 一致性。也可以开源出去，让更多的人使用。

本文将从零开始，一步步带你写一个基于 Vue3 的组件库，希望能帮助到你。

一、准备工作 ​

首先确定项目的技术栈，本项目采用 pnpm + Vue3 + TypeScript + Vite + Postcss + ESLint + Prettier。

二、项目初始化 ​

1. 创建项目 ​

使用 Vite 创建项目，执行以下命令：

bash
npm create vue@3

5. 配置 postcss ​

安装相关依赖包

bash
pnpm i postcss-each-variables postcss-cli postcss-nested postcss-each postcss-for postcss-color-mix -D
pnpm i postcss-each-variables postcss-cli postcss-nested postcss-each postcss-for postcss-color-mix -D

在项目根目录下创建 postcss.config.cjs 文件，内容如下：

js
/* eslint-env node */
module.exports = {
  plugins: [
    require('postcss-each-variables'),
    require('postcss-nested'),
    require('postcss-each')({
      plugins: {
        beforeEach: [
          require('postcss-for'),
          require('postcss-color-mix')
        ]
      }
    }),
  ]
}
/* eslint-env node */
module.exports = {
  plugins: [
